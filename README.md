## Description

[Nest](https://github.com/nestjs/nest) API for a basic crud on products. 
  
## Installation

```bash
$ git clone
$ npm install
```
## Update .env PORT and MONGO_URI at root level with your own credentiales
```bash
PORT=YOUR_PORT
MONGO_URI=YOUR_MONGO_ATLAS_URI
```
## Start for development

```
$ npm run start
```
